(1) Scope of var, let, and const:

var is function-scoped.

let and const are block-scoped.

const cannot be reassigned.

(2) Use Cases of null and undefined::

Use null to represent intentional absence of a value.

Use undefined for uninitialized variables or accessing non-existent properties.

const cannot be reassigned.

(3) REST API (Representational State Transfer API):

A set of rules for building web services.

Stateless, uses URLs to access resources.

Uses HTTP methods (GET, POST, etc.) to interact with resources.

Utilizes HTTP status codes for response information.
